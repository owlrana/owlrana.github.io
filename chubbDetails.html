Automated Noise Removal


Open Source Contribution @GirlScript Foundation (GSSoC '21)

My contribution is merged into Rotten Scripts and can be found here.


The fundamental aim of this script was to automate the process of Audio Extraction, Profiling and Cleaning before the Mixing Stage for Voice-Overs or Audio Recordings in videos, with a script that is adjustable according to the user using Python. This is achieved with the help of the following open-sourced softwares:

  • FFmpeg
  • Sox
  • Python

    • This script decreases the time required to denoise files from 1 hour to less than 2 minutes (avg. length of 5 minutes, for a batch of 20 files)


    • Language: Python / Scripting
    • Category: Open Source
