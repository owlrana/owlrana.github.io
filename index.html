Automated Noise Removal


Open Source Contribution @GirlScript Foundation (GSSoC '21)

My contribution is merged into Rotten Scripts and can be found here.


The fundamental aim of this script was to automate the process of Audio Extraction, Profiling and Cleaning before the Mixing Stage for Voice-Overs or Audio Recordings in videos, with a script that is adjustable according to the user using Python. This is achieved with the help of the following open-sourced softwares:

  • FFmpeg
  • Sox
  • Python

    • This script decreases the time required to denoise files from 1 hour to less than 2 minutes (avg. length of 5 minutes, for a batch of 20 files)

    Visual Identity


    Brand Visual Identity for Code for Cause

    Some of the graphics created during the term can be found here.


    Distinct Visual Identity for Code for Cause from the ground-up.
    To have a disctinct Visual Style in the world of ever-growing edu-startups and their ever-changing products, it would mean to not only have control over the Logo & Posters, but also each graphic, each colour, each font and the size of fonts used while keeping new offerings fresh & attractive.

    It is hard to have such a control without a guideline as each designer is unqiue and brings in a different culture/style.

    Enter Brand Identity Guidelines.

    A Brand Identity Guideline ensures a constant control and can be referred to by future employees to make graphics without straying away from the original vision for the Brand's Visuals. This guide is different for each brand and should be created while keeping the Vision behind the brand in mind and paying attention to the story they want to showcase.

    In case of Code for Cause:

  • CFC should be visually distinct from its competitors.
  • CFC should showcase a factor of trust in each offering.
  • CFC's Graphics should be more fun, up-beat & attractive than competitors.
  • CFC should convey a story of Passion towards Coding rather than dreams of Jobs.
  • CFC should support imagination, spirituality and freedom of ideas.
